a clean and intuitive ui

Fringe uses a minimalistic approach in its design, putting ease of use and responsiveness first while focusing on clean typography and content.

Fringe Messenger helps you focus only on what's important to you by keeping its interface really simple and minimalistic so you never have to search for things.

Fringe provides real-time synchronization of all your data between all your devices using its cloud based architecture.

Fringe's proprietary encryption system - The BlackBox, has been built into every client app. The BlackBox guarantees safety even against a Shor's algorithm brute-force attack by a Quantum computer.

All profile data is written to our servers using a proprietary file system called the BlackHose File System (BFS) which is a type of deniable encryption. This data cannot be read by us through mathematical analysis or physical disk inspection.

Fringe uses a form of JWT based stateless authentication to prevent hackers from abusing users' login credentials.

Fringe uses 'personal' and 'work' spaces so you can segregate your casual and professional conversations.
